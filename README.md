# 💻AutoRepo
## 🚀 소개
### 프로젝트 소개
깃허브 연동으로 딸깍 한 번이면 세팅이 끝나는
레포 세팅 관련한 기능들을 자동화 해주는 서비스 AutoRepo입니다!


### 주요 기능
- ISSUE/PR 템플릿 생성 및 관리
- 라벨 생성 및 관리
- README 자동 생성
- 템플릿 대시보드

### 기술적 특징
OPEN AI를 활용한 리드미 자동 생성

## 💻 설치 방법
1. 저장소 복제
```bash
git clone https://github.com/hysong4u/AutoRepo
cd AutoRepo
```

2. 프로젝트 빌드
```bash
./gradlew build
```

3. 애플리케이션 실행
```bash
./gradlew bootRun
```

4. JAR 파일 실행 (선택사항)
```bash
java -jar build/libs/AutoRepo-0.0.1-SNAPSHOT.jar
```
